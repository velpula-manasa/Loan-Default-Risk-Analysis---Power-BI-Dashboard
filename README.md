# Loan-Default-Risk-Analysis---Power-BI-Dashboard
This project analyzes borrower financial data to identify key factors influencing loan default risk, helping financial institutions make better lending decisions and reduce potential losses.

## Technologies Used
- Power BI
- SQL Server
- Power Query
- DAX
- Microsoft Power BI

## Project Description
Loan data was imported from SQL Server into Power BI for analysis and visualization. A dataflow and Enterprise Gateway were configured to create a secure connection between the database and Power BI, allowing the data to be accessed and used for reporting.

Power Query was used to clean and transform the raw dataset. Tasks included handling missing values, correcting data types, standardizing formats, and organizing borrower information properly. These steps helped ensure that the data was accurate and reliable for analysis.

After preparing the data, custom DAX measures and calculated columns were created to calculate important metrics related to borrower financial profiles and loan performance. These calculations helped analyze borrower income levels, debt-to-income ratios, credit scores, and other financial indicators.

An interactive Power BI dashboard was developed to present the analysis in a clear and meaningful way. The dashboard includes charts, filters, and slicers that allow users to explore the data easily and understand relationships between borrower characteristics and loan default risk.

## Key Insights
- Higher debt-to-income (DTI) ratios are associated with increased loan default risk.
- Borrowers with lower credit scores show a higher probability of default.
- Employment type and income level influence loan repayment behavior.
- Mortgage status and number of dependents also impact default patterns.

These insights help identify high-risk borrower profiles and support data-driven decision-making in loan approval and risk management.
